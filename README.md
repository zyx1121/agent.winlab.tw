# Agent Portal

Web portal for WinLab AI Agent — manage and interact with the multi-platform bot.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Backend**: Supabase
- **Styling**: Tailwind CSS + shadcn/ui
- **Package Manager**: Bun

## Getting Started

```bash
bun install
bun dev
```
